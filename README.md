# Portfolio Website

This is a personal portfolio website for Sambasivarao, designed to showcase skills, experience, projects, certifications, and contact information. The site is built using only HTML and CSS, with a modern, responsive layout and a focus on clean design and usability.

## Project Structure

- **index.html**: Landing page with a hero section introducing Sambasivarao, a brief description, and navigation to other sections.
- **About-Me.html**: Details about Sambasivarao, including a bio, skills, and fun facts.
- **resume.html**: Education and experience timeline, with a downloadable resume.
- **Portfolio.html**: Showcases selected projects with images and categories.
- **certifications.html**: Displays certifications, awards, and hackathon achievements.
- **contact.html**: Contact information including email, phone, and location.

### CSS Files
- **style.css**: Main styles for the landing page (index.html).
- **About-Me.css**: Styles for the About Me section.
- **resume.css**: Styles for the Resume section.
- **Portfolio.css**: Styles for the Portfolio section.
- **certifications.css**: Styles for the Certifications section.
- **contact.css**: Styles for the Contact section.

### Assets
- **images/**: Contains profile and project images.
- **icons/**: SVG icons for navigation and UI elements.
- **documents/**: Contains downloadable files such as the resume PDF.

## Features
- **Responsive Design**: Works well on desktops, tablets, and mobile devices.
- **Navigation Sidebar**: Fixed sidebar for easy navigation between sections.
- **Modern UI**: Uses Google Fonts (Montserrat, Oswald) and custom icons.
- **No JavaScript**: All interactivity and layout are handled with HTML and CSS only.

## Customization
- Replace images in the `images/` folder with your own profile and project images.
- Update the text content in each HTML file to reflect your own experience, projects, and contact details.
- Edit the CSS files to change colors, fonts, or layout as desired.
- Add new sections or pages by following the existing structure.

## Usage
1. Open `index.html` in your browser to view the site.
2. Navigate using the sidebar or navigation icons.
3. Download the resume from the Resume section.
